1. Main Form & Administrative Security

The Student Challenge:
We needed to create a single gateway screen that lets general buyers jump right to current tickets without roadblocks, but simultaneously secures the admin entry path from random client intervention.
How I Coded It:
I utilized InputBox prompts that process inputted values into localized strings. By passing the inputs securely using parameter mapping (@email and @password), I prevented SQL Injection vulnerability and intercepted unwanted access before bringing up the dashboard.
' Code Snippet from Project: Intercepting admin access with secure parameter mapping Dim email As String = InputBox("Enter Admin Email:", "Security Check") Dim password As String = InputBox("Enter Admin Password:", "Security Check") Dim cmd As New MySqlCommand("SELECT * FROM Admin WHERE Email=@email AND Password=@password", con) cmd.Parameters.AddWithValue("@email", email) cmd.Parameters.AddWithValue("@password", password) con.Open() Dim reader As MySqlDataReader = cmd.ExecuteReader() If reader.HasRows Then Admin.Show() Else MessageBox.Show("Incorrect email or password.", "Access Denied") End If

4. Multi-Form State & Tier Processing

The Student Challenge:
When users select a particular localized seating bracket (VIP, Premium, or various Standard layouts), the program needs to retain these values across separate windows without dropping variable contexts.
How I Coded It:
I authored structural form properties. When user interaction triggers the submission switch, the program checks conditional radio values, maps pricing properties to internal integer configurations, and transfers the state objects explicitly to the upcoming module instance.
' Code Snippet from Project: Forwarding runtime values directly into explicit object fields Dim f As New BuyTicket With { .SelectedArtist = SelectedArtist, .ConcertDate = ConcertDate, .VIP_Price = CInt(VIP_P), .Prem_Price = CInt(Prem_P), .Std_Price = CInt(Std_P), .PreviousForm = Me } If rdVIP.Checked Then : f.SelectedTier = "VIP" ElseIf rdPremium.Checked Then : f.SelectedTier = "Premium" End If

5. Ticket Pricing Pipeline & Seat Allocation

The Student Challenge:
Calculating accurate sub-totals, imposing regional service taxes, and generating specific, non-overlapping alpha-numeric seating IDs systematically based on quantity demands.
How I Coded It:
I established strict calculation pipelines using explicit casting operators. The total billing statement multiplies quantities against selected properties, applies a 0.06 floating multiplier for service tax, and utilizes a mathematical base offset loop to auto-generate seat labels.
' Code Snippet from Project: Pricing calculation and incremental seat array allocation Dim subtotal As Integer = qty * price Dim tax As Double = subtotal * 0.06 Dim totalAll As Double = subtotal + tax Dim seats As New List(Of String) For i As Integer = 1 To qty If SelectedTier = "VIP" Then seats.Add("V" & (233 + i)) ' Auto-indexes unique seat ranges End If Next

6. Database Persistence & Receipt Generation

The Student Challenge:
Capturing the final checkout values including randomized Customer IDs, explicit purchase timestamps, seat string collection arrays, and transactional pricing and committing them into database records.
How I Coded It:
I compressed arrays using a String.Join function to store seat selections as a single delimited column string. Then, using an active connection scope block, I map individual inputs to transactional table values, preserving records securely inside the database structure.
' Code Snippet from Project: Compressing lists and executing database persistence logs Dim seatText As String = String.Join(",", SeatNumbers) Dim query As String = "INSERT INTO receipt (OrderID, BuyTime, ConcertName, Tier, Quantity, SeatNumber, UnitPrice, Subtotal, Tax, TotalPrice) VALUES (@orderID, @buyTime, @concertName, @tier, @qty, @seats, @unitPrice, @subtotal, @tax, @total)" Using cmd As New MySqlCommand(query, con) cmd.Parameters.AddWithValue("@orderID", lblCustomerId.Text) cmd.Parameters.AddWithValue("@seats", seatText) cmd.Parameters.AddWithValue("@total", TotalAll) cmd.ExecuteNonQuery() End Using
